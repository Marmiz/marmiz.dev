# A statically generated blog using Next.js, Markdown, and TypeScript.

This is my personal blog website, deployed and hosted on [Vercel](https://vercel.com).

The blog posts are stored in `/_posts` as Markdown files with front matter support.

It uses [Tailwind CSS](https://tailwindcss.com) for the styling. To control the generated stylesheet's filesize, this uses Tailwind CSS' v2.0 [`purge` option](https://tailwindcss.com/docs/controlling-file-size/#removing-unused-css) to remove unused CSS.

Feel free to reach out with any question or comments.

## Notes

This blog was created from this [starter](https://github.com/vercel/next.js/tree/canary/examples/blog-starter-typescript).
